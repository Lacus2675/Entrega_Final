# Entrega_Final
Proyecto Entrega final

Modulos a instalar :

sqlite3 (para las bases sqlite)
os (uso para borrar pantalla)
colorama (color en la consola)
pathlib (al buscar un archivo directamente tube problemas , investigue y este modulo permite buscarlos en cualquier patch sin ningun problema)

Recomendacion :

Correr desde consola windows cmd o terminal en pantalla completa , dado que la consola se puede agrandar o poner en
modo pantalla completa esto es mejor para que no se corte en las vistas por el largo del registro de algun campo

Extructura :

Base_De_Datos -> dentro tiene las db 

Informes_Stock_Bajo -> dentro tiene el informe que puedo exportar de stock bajo

 -- main -> archivo inicial 
 
 -- comexion_db.py -> funciones de conexion a base de datos
 
 -- modulos.py -> funciones de la base de datos
 
 -- db.txt -> dentro en la primer linea esta el nombre de la db a usar , cambiando el nombre de la db que estan dentro de la carpeta
    Base_De_Datos puedo cambiar la db
 
 -- ejecutar_opcional.bat -> ejecuta el sistema en pantalla completa
